______________________________________________________________________

UI Controls:

Left side: Log File controls

Add File: Click the Add button and select a file (.csv) to add to the application.

Remove File: select a log file from the LogFile list then click the remove Button.

Resize: resize the Log File area by dragging the divider between the LogFile display and Events display

______________________________________________________________________

Event Table:

Search:  use the drop down to select the column to search on.  Type in your search string

Sort: Click on the column headers to sort them ascending or descending.  Triple click to return to the original order.

Resize Columns: Change column sizes by dragging the column dividers

Expand Details:  Double click a row to view expanded details in a separate pop-up window.
